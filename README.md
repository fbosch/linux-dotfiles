# Dotfiles


## Installation

* Add the following alias to your shell config:
```sh
alias config='/usr/bin/git --git-dir=$HOME/.cfg --work-tree=$HOME'
```

* Clone the repository
```sh
git clone --bare git@github.com:fbosch/dotfiles.git $HOME/.cfg
```
